# 电力负荷预测系统 (Power Load Prediction)

基于XGBoost集成学习算法的电力负荷预测项目，用于预测未来时刻的电力负荷值。

## 项目简介

本项目利用机器学习技术对历史电力负荷数据进行分析和建模，通过XGBoost回归模型实现电力负荷的精准预测。项目包含完整的数据预处理、特征工程、模型训练、评估和预测流程。

## 使用说明

### 1. 数据准备

将训练数据和测试数据分别放置在 `data/` 目录下：
- `train.csv`: 用于模型训练的历史电力负荷数据
- `test.csv`: 用于模型测试和预测的数据

数据格式要求：CSV文件应包含 `time`（时间戳）和 `power_load`（电力负荷值）两列。

### 2. 模型训练

运行训练脚本以构建预测模型：

```bash
cd src
python train.py
```

训练过程包括：
- 数据预处理（时间格式化、排序、去重）
- 特征工程（时间特征提取、滞后特征构造）
- 网格搜索与交叉验证寻找最优超参数
- 模型评估（MSE、RMSE、MAE、MAPE）
- 模型保存至 `model/model.pkl`

### 3. 模型预测

使用训练好的模型进行预测：

```bash
cd src
python predict.py
```

预测流程：
- 加载预训练模型
- 解析预测时间点的特征
- 执行预测并生成结果
- 可视化实际值与预测值对比图
- 记录预测性能指标到日志

## 核心功能

### 特征工程

项目实现了丰富的特征提取策略：

1. **时间特征**:
   - 小时特征（24维One-Hot编码）
   - 月份特征（12维One-Hot编码）

2. **滞后特征**:
   - 前1小时负荷
   - 前2小时负荷
   - 前3小时负荷
   - 昨日同时刻负荷

### 模型评估指标

- **均方误差 (MSE)**: 衡量预测值与真实值的平方差均值
- **均方根误差 (RMSE)**: MSE的平方根，单位与原数据一致
- **平均绝对误差 (MAE)**: 预测值与真实值绝对差的均值
- **平均绝对百分比误差 (MAPE)**: 相对误差的百分比表示

### 日志系统

项目内置完善的日志记录机制：
- 训练日志：记录模型训练过程中的关键信息
- 预测日志：记录预测执行过程和性能指标
- 日志文件按日期命名，便于追踪和管理

## 可视化分析

项目提供两种可视化图表：

1. **电力负荷数据分析图**: 
   - 负荷整体分布直方图
   - 各小时平均负荷趋势
   - 各月份平均负荷趋势
   - 工作日与周末负荷对比

2. **预测结果图**:
   - 实际负荷曲线
   - 预测负荷曲线
   - 两者对比分析

## 注意事项

1. **时间序列特性**: 本项目采用时间顺序划分数据集（80%训练集，20%测试集），避免随机划分导致的数据泄露问题。

2. **特征一致性**: 预测时的特征提取必须与训练时保持一致，确保模型输入格式正确。

3. **缺失值处理**: 对于历史数据中不存在的时刻，使用默认值500进行填充。

4. **中文显示**: 图表已配置SimHei字体以支持中文标签正常显示。
